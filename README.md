# desafio-copa

## Instalar dependencias 
```
yarn install
```

### Iniciar Projeto
```
yarn serve
```

### Iniciar back-end
```
yarn backend
```

### Detalhes do projeto
```
Sistema criado para simular jogos da copa do mundo, permitindo 
ao usuário cadastrar os times e simular as próximas partidas.

Desenvolvido com:
- Axios
- Vue 3
- Vue-router
- API ruby on rails
```
